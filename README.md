# Numina Scrapers

TypeScript/Node.js service for scraping and aggregating fitness class data from various providers.

## Overview

This service scrapes group fitness class data from gyms, studios, and booking platforms, then pushes it to the Numina backend API. It's designed with a modular provider adapter pattern for easy addition of new data sources.

## Target Providers

- Mindbody
- ClassPass
- Equinox
- 24 Hour Fitness
- Local gym chains
- Boutique studios

## Technology Stack

- **Language**: TypeScript (Node.js 20+)
- **Automation**: Puppeteer
- **Database**: SQLite (operations tracking)
- **HTTP Client**: Fetch API
- **Testing**: Jest
